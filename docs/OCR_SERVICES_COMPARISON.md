| **项目地址** | https://github.com/tesseract-ocr/tesseract |

**系统要求**：
```bash
# 1. 安装Tesseract
apt-get install tesseract-ocr tesseract-ocr-chi-sim

# 2. 安装Python封装
pip3 install pytesseract pillow

# 3. 测试识别
tesseract image.jpg stdout

# 4. Python调用
from pytesseract import image_to_string
text = image_to_string(Image.open('image.jpg'))
```

| **劣势** | - 首次识别需要加载模型<br>- 占用内存较大<br>- 速度不如Tesseract（小图） |

**系统要求**：
```bash
# 1. 安装EasyOCR
pip3 install easyocr

# 2. Python调用
from easyocr import Reader
reader = Reader(['ch_sim', 'en'], gpu=False)  # 中文+英文

# 3. 识别图片
result = reader.readtext('image.jpg')
print(result)
```
